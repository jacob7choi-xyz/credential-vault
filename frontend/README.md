# Credential Vault Frontend

Next.js 15 Web3 dashboard for the Credential Vault platform. Connects to the three-contract blockchain layer for identity management and credential verification.

## Tech Stack

- Next.js 15 (App Router, Turbopack)
- React 19
- TypeScript (strict mode)
- Wagmi v2 + viem for contract interactions
- RainbowKit for wallet connection
- Tailwind CSS v4
- Framer Motion for animations

## Setup

```bash
npm install
npm run dev
```

Requires a running Hardhat node with deployed contracts. See the root README for full local stack instructions.

## Structure

```
src/
  app/
    page.tsx           # Main dashboard
    layout.tsx         # Root layout
    providers.tsx      # Web3 providers (Wagmi + RainbowKit + React Query)
    globals.css        # Cyberpunk terminal styling
  hooks/
    useContracts.ts    # Custom Wagmi hooks for all contract interactions
config/
  wagmi.ts             # Chain and wallet configuration
  contracts.json       # Contract addresses + ABIs (auto-generated by deploy script)
```

## Current Capabilities

- DID creation with modal form
- Issuer registration (admin only)
- Credential issuance (authorized issuers only)
- Credential display with status badges (VALID/REVOKED)
- Stats grid (DID status, credential count, issuer authorization)
- Quick verify via contract read
